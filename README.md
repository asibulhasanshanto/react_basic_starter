# React boilerplate

## Features

- Authentication
- Authorization
- Dark mode
- Toasts

## Description

- The Authentication, Authorization and Dark mode features are implemented using React Context API.
- The Toasts feature is implemented using react-hot-toast.
- The Login form is validated using [Formik](https://formik.org/docs/api/useFormik) and [Yup](https://formik.org/docs/guides/validation).
- As ui styling library, TailwindCSS is used. And a package named 'prettier-plugin-tailwindcss' along with prettier is used to format the tailwind classes.

## Installation

```bash
yarn install
yarn dev
```
